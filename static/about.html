OUR DATA

SpotCrime
Crime data was collected from the SpotCrime API by providing lat long coordinates and a given radius. To do so, we aggregated unique results provided by multiple queries. Queries were made in increments of .015 degrees and a radius of .02 degrees.

US Census Bureau
We wrote a script that does GET requests to the census bureau's public API to gather information about various demographics for each zip code including total population, population by economic status, population by race, population by age, and other factors
