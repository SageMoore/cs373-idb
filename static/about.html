OUR DATA

SpotCrime
Crime data was collected from the SpotCrime API by providing lat long coordinates and a given radius. Because SpotCrime limits the results for each query, our team decided to break up each search into multiple subqueries and collect the unique results. To do so, we searched for a grid in increments of .015 degrees and then search by radius of .02 degrees. This causes some overlap, but is necessary since the circular search areas from the radii could lead to gaps in our data.

US Census Bureau
We wrote a script that does GET requests to the census bureau's public API to gather information about various demographics for each zip code including total population, population by economic status, population by race, population by age, and other factors. The factors were found by iterating through a JSON file that mapped census variables such as B01001_001E to their corresponding descriptions (such as total population)
